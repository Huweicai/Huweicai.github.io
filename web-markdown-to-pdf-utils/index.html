Markdowntopdf

地址：https://www.markdowntopdf.com/

评价：转出来效果还是可以的，但是只能通过文件转成 PDF，不能直接在线编辑复制粘贴，稍稍麻烦一点。

效果图：

md2pdf

地址：https://md2pdf.netlify.app/

评价：可以在线编辑，还是开源的，但是会有水印。

效果图：

冷熊简历

地址：http://cv.ftqq.com/?fr=github#

评价：支持在线编辑，偏简历制作，格式比上面两个要精致一点，不过分隔符是居中的，但 Markdown 文本基本都是左对齐的，所以会看着稍稍有点奇怪。 效果图：

dillinger

地址：https://dillinger.io/

评价：网站风格很有设计感，同时支持导出多种格式，但最大的遗憾是中文 pdf 会乱码，只支持英文文档。

效果图：

coolaf

地址：http://coolaf.com/tool/md

评价：这是一个工具集网站，支持导出多种格式，导出结果样式也中规中矩。

效果图：

总结

目前总的来看效果和体验的话：Markdowntopdf 应该是最推荐的，如果纯英文文档的话可以考虑 dillinger，dillinger 能支持中文的话就完美了。
